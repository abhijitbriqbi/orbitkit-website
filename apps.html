Setup Sequence

T‑minus 4 steps to live data

From monday.com to Power BI or Tableau in under five minutes.

Install from marketplace

Add OrbitKit BI from the monday.com App Marketplace. One-click OAuth 2.0 — no account creation beyond monday.com.

Feature manifest
🔐

monday.com OAuth 2.0

Authorize once. AES-256 encrypted token storage. Never exposed client-side. Revoke anytime from monday.com settings.

⚡

Live JSON feed

Every BI refresh hits our endpoint which proxies to monday.com's GraphQL API in real time. Zero caching, always current.

🧩

Multi-board merge

Combine multiple boards into one connector. A board_name column is added automatically for slicing in your BI tool.

🎯

Visual filter builder

Build row-level filters in the wizard UI — contains, equals, greater than, is empty. Applied server-side before data reaches BI.

🔑

Tokenized feed URLs

Token in query param means Power BI "Get Data → Web" uses Anonymous auth. One URL, no dialogs, no credential management.

📋

Connection management

My Connections tab: view credentials, regenerate tokens, edit configs, verify OAuth status. Full control in one place.
