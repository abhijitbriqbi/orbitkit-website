Create a connection in OrbitKit BI and copy the Feed URL. In Power BI: Home → Get Data → Web. Paste the URL, choose Anonymous authentication. Select your tables and Load.
Create a Tableau connection and copy the WDC URL. In Tableau: Connect → Web Data Connector. Paste the WDC URL, enter your access token from View Credentials, click Connect.
